# ExpressKT

Kotlin wrappers for [ExpressJS](http://expressjs.com/) library.

## How to use

Compile source :

```
// Install npm dependencies
./gradlew demo:yarn_install
// Build the JS file
./gradlew demo:build
```

Run node server :
```
node demo/output/index.js
```

Test server at https://localhost:3000/test.
You should see Hello world!
